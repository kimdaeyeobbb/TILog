## 6. 다음 명령의 의미를 기술하시오.


- cd ..         
  - 현재 디렉토리의 부모 디렉토리로 이동함


- cd ../temp
  - 상대경로를 사용하여 현재 디렉토리의 상위 디렉토리로 이동 후 temp 디렉토리로 이동함


- cd ~user2
  - user2는 사용자명에 해당되며, 입력한 사용자의 홈 디렉토리로 이동함.


- cd ~/temp
  - ~명령어를 통해 자신의 홈 디렉토리로 이동후 temp다렉토리로 이동 (홈에 있는 temp로 이동)



- cd /tmp
  - 절대경로를 사용하여 /tmp 디렉토리로 이동
